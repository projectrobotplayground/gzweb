View SDF from URL

This example shows how to visualize an SDF file on the browser by passing the URLs of its resources.

The SDF parsing happens completely on the browser, so there is no need for a GzWeb server to be runnning in the backend.

You can test it as follows:

  1. Under "Resources URLs", type the URL to a resource needed by the SDF (like meshes, textures, etc). If no resources are missing, skip to step 4.
  2. Click "Add URL"
  3. Go back to step 1
  4. Under "SDF URL", fill the full URL to the SDF file
  5. Click "Load SDF!"
